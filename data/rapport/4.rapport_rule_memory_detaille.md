# Rapport détaillé — Brique `RuleMemory`

## 1. Objectif de la brique

`RuleMemory` représente la mémoire **procédurale** des agents.

Elle contient :
- règles de fonctionnement
- contraintes
- politiques d’exécution
- conventions techniques
- règles de sécurité

👉 Contrairement aux autres mémoires :
- elle est **prioritaire**
- elle est **toujours injectée**
- elle influence directement le comportement de l’agent

---

## 2. Décisions d’architecture retenues

### 2.1 Nature des règles

Chaque règle est une **instruction explicite et structurée**, pas un simple texte libre.

Exemples :
- “Ne jamais exécuter une commande destructive sans validation utilisateur”
- “Toujours utiliser @ConfigurationProperties dans les projets Spring”
- “Lire le dossier Shared avant d’utiliser un tool”

---

### 2.2 Scope

```text
- GLOBAL (shared entre tous les agents)
- AGENT (spécifique à un agent)
- PROJECT (lié à un projet)
```

---

### 2.3 Priorité

Chaque règle possède un **niveau de priorité** :

```text
- CRITICAL
- HIGH
- MEDIUM
- LOW
```

👉 Permet :
- résolution de conflits
- tri avant injection
- contrôle du prompt

---

### 2.4 Stockage V1

👉 Base SQL (PostgreSQL recommandé)

Pourquoi :
- structuré
- audit
- versionnable plus tard
- filtrable

---

### 2.5 Injection

👉 Chargement **systématique** dans le prompt

Mais :
- trié par priorité
- limité en taille

---

## 3. Risques et arbitrages

### 3.1 Règles contradictoires

Ex :
- “Toujours exécuter automatiquement”
- “Toujours demander validation”

👉 Arbitrage :
- priorité
- scope (AGENT > GLOBAL)
- possibilité de désactivation

---

### 3.2 Surcharge du prompt

👉 Arbitrage :
- limite max règles injectées
- tri par priorité
- regroupement possible

---

### 3.3 Règles obsolètes

👉 Arbitrage :
- statut (ACTIVE / DISABLED)
- mise à jour manuelle

---

```java

    DISABLED
}
```

---

## 5.2 Entité

```java
@Entity
@Table(name = "rule_memory")
public class RuleEntry {

    @Id
    @GeneratedValue
    private Long id;

    private String agentId;

    @Enumerated(EnumType.STRING)
    private RuleScope scope;

    private String scopeId;

    private String title;

    @Column(length = 5000)
    private String content;

    @Enumerated(EnumType.STRING)
    private RulePriority priority;

    @Enumerated(EnumType.STRING)
    private RuleStatus status;

    private Instant createdAt;
    private Instant updatedAt;
}
```

---
