# Rapport détaillé — Brique `ContextAssembler`

## 1. Objectif de la brique

`ContextAssembler` est la brique **centrale** qui construit le **contexte final envoyé au LLM**.

👉 Elle orchestre toutes les mémoires :
- `RuleMemory` (prioritaire)
- `ConversationMemory` (court terme)
- `SemanticMemory` (long terme pertinent)

👉 Elle garantit :
- cohérence
- pertinence
- taille maîtrisée

---

## 2. Rôle dans l’architecture

```text
User input
    → ConversationMemory (update)
    → ContextAssembler
        → RuleMemory
        → ConversationMemory
        → MemoryRetriever (SemanticMemory)
    → LLM
```

👉 C’est le **point de convergence** de tout ton système de mémoire.

---

## 3. Décisions d’architecture retenues

### 3.1 Composition stricte du contexte

Ordre imposé :

```text
1. Rules (RuleMemory)
2. Relevant Semantic Memories
3. Conversation (résumé + récents)
4. User message (dernier)
```

👉 Important pour :
- guider le LLM
- éviter ambiguïtés

---

### 3.2 Construction déterministe

👉 Pas de magie en V1 :

- pas de LLM dans l’assembler
- pas de heuristique complexe
- pas de dynamique adaptative

---

### 3.3 Limitation stricte

Le contexte doit respecter :

```text
- maxCharacters
- maxMemories
- maxRules
```

---

## 6.3 Implémentation

```java
public class DefaultContextAssembler implements ContextAssembler {

    private final RuleService ruleService;
    private final ConversationMemoryService conversationMemoryService;
    private final MemoryRetriever memoryRetriever;

    private final int maxMemories = 10;
    private final int maxRules = 10;
    private final int maxCharacters = 15000;

    public DefaultContextAssembler(
            RuleService ruleService,
            ConversationMemoryService conversationMemoryService,
            MemoryRetriever memoryRetriever
    ) {
        this.ruleService = ruleService;
        this.conversationMemoryService = conversationMemoryService;
        this.memoryRetriever = memoryRetriever;
    }

    @Override
    public String buildContext(ContextRequest request) {

        StringBuilder context = new StringBuilder();

        // 1. Rules
        var rules = ruleService.getApplicableRules(request.agentId(), request.projectId())
                .stream()
                .limit(maxRules)
                .toList();

        context.append("## Rules\n");
        for (var rule : rules) {
            context.append("- [").append(rule.getPriority()).append("] ")
                   .append(rule.getContent()).append("\n");
        }

        // 2. Semantic Memory
        var memories = memoryRetriever.retrieve(
                new MemoryQuery(
                        request.agentId(),
                        request.projectId(),
                        request.userMessage(),
                        Set.of(MemoryScope.AGENT, MemoryScope.SHARED, MemoryScope.PROJECT),
                        Set.of(MemoryType.FACT, MemoryType.CONTEXT, MemoryType.DECISION),
                        maxMemories
                )
        );

        context.append("\n## Relevant Memories\n");
        for (var memory : memories) {
            context.append("- ").append(memory.getContent()).append("\n");
        }

        // 3. Conversation
        String conversation = conversationMemoryService.buildContext(
                new ConversationMemoryKey(request.agentId(), request.conversationId())
        );

        context.append("\n## Conversation\n");
        context.append(conversation);

        // 4. User message
        context.append("\n\n## User Input\n");
        context.append(request.userMessage());

        return trim(context.toString());
    }

    private String trim(String context) {
        if (context.length() <= maxCharacters) {
            return context;
        }
        return context.substring(context.length() - maxCharacters);
    }
}
```
