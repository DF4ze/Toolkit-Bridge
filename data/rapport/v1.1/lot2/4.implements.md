Objectif de la phase 4 :
- extraire le bloc agents/runtime/policy/tools dans `AdminAgentQueryService`
- conserver `TechnicalAdminFacade` comme façade délégante
- conserver `getOverview()` dans la façade
- conserver strictement les contrats DTO et contrôleurs
- ne pas modifier le domaine runtime/policy/tools au-delà de l’extraction admin

- `AdminAgentQueryService`

Implémentation attendue :

## A. Création de `AdminAgentQueryService`
Créer un service Spring simple dans `service.admin`.

Dépendances attendues :
- `AgentDefinitionService`
- `AgentRuntimeRegistry`
- `AgentPolicyRegistry`
- `ToolRegistryService`

Responsabilités :
Déplacer dans ce service, sans changement de comportement observable :
- `listAgents()`
- `toAgentItem(...)`
- `resolvePolicyWithoutRuntime(...)`
- `resolveExposedToolsFromPolicy(...)`
- `toRuntimeItem(...)`
- `toPolicyItem(...)`
- `normalize(...)` si encore nécessaire localement

Contraintes :
- conserver strictement la différence de comportement entre :
    - runtime présent
    - runtime absent avec fallback policy/tools
- conserver l’ordre actuel des `exposedTools` quand runtime présent
- conserver le tri/filtrage actuel des `exposedTools` quand runtime absent
- conserver `runtime == null` dans `AgentItem` si runtime absent
- conserver le fallback `availability = "UNKNOWN"` si `availability` est null
- conserver le mapping `accessibleMemoryScopes` en `Enum::name`
- ne pas transformer ce service en moteur métier policy/tool
- garder le service comme query/projection admin

## B. Adaptation de `TechnicalAdminFacade`
Modifier `TechnicalAdminFacade` pour :
- injecter `AdminAgentQueryService`
- conserver la méthode publique `listAgents()`
- transformer `listAgents()` en simple délégation
- conserver `getOverview()` en structure inchangée
- faire en sorte que `getOverview()` continue d’appeler `listAgents()` via la façade ou une délégation propre, sans changement de contrat

Nettoyage attendu :
- retirer de la façade les dépendances/imports du bloc agents déplacé
- retirer de la façade les mappings/helpers déplacés
- conserver dans la façade uniquement ce qui reste encore de son rôle d’orchestration globale
